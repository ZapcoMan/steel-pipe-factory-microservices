## 🔧 开发环境配置

### 前置要求
- Node.js >= 14.0.0
- 微信开发者工具
- pnpm (推荐) 或 npm

### 安装步骤
```bash
# 克隆项目
git clone <项目地址>

# 安装依赖
pnpm install

# 启动开发模式
pnpm run dev:mp-weixin

# 构建生产版本
pnpm run build:mp-weixin
```

### API 地址配置
编辑 `.env` 文件:
```env
VITE_API_BASE_URL=http://你的服务器地址：端口
VITE_TIMEOUT=60000
```

## ⚠️ 重要提示

### API 调用规范
- ✅ 所有请求必须包含错误处理
- ✅ 建议使用 try-catch 包裹
- ✅ 提供用户友好的错误提示
- ✅ 准备好模拟数据作为降级方案

### 环境变量
- ⚠️ 只有以 `VITE_` 开头的环境变量才会暴露给客户端
- ⚠️ 修改环境变量后需要重启开发服务器

### 图片资源
- 📷 本地图片放在 `static/` 目录
- 🔗 使用绝对路径引用：`/static/logo.png`
