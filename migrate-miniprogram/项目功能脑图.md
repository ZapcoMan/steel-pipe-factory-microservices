## 🔌 API 调用流程

```
页面加载
    ↓
调用 API 方法
    ↓
HttpRequest 封装
    ├─→ 请求拦截器
    │   ├─ 添加 Content-Type
    │   └─ 添加 Token(可选)
    ↓
发送请求到服务器
    ↓
服务器响应
    ↓
响应拦截器
    ├─ 200/201 → 返回数据 ✓
    ├─ 401 → 未授权提示 ✗
    ├─ 403 → 拒绝访问 ✗
    ├─ 404 → 资源不存在 ✗
    └─ 500 → 服务器错误 ✗
    ↓
成功 → 更新页面数据
失败 → 显示模拟数据 + 错误提示
```

## 📊 数据流向

```
环境变量 (.env)
    ↓
配置文件 (src/config/index.ts)
    ↓
HttpRequest 实例 (src/utils/request.ts)
    ↓
API 服务层 (src/api/*.ts)
    ├─ api.ts (通用服务)
    ├─ homeApi.ts (首页)
    ├─ steelPriceApi.ts (价格)
    ├─ techDataApi.ts (技术资料)
    └─ sampleApi.ts (样品)
    ↓
页面组件 (src/pages/*.vue)
    ↓
用户界面展示
```

## 🔄 状态管理

采用 Vue 3 Composition API 的 ref 和 reactive:

```typescript
// 示例：价格页面
const priceList = ref<SteelPrice[]>([]);  // 价格数据
const selectedType = ref(0);              // 选中类型
const activeBarIndex = ref(-1);           // 激活柱状图

// 异步加载
const loadPriceData = async () => {
  const data = await steelPriceApi.getSteelPriceData();
  priceList.value = data;
};
```

## 🎯 核心功能实现

### 1. 容错机制
所有 API 调用都包含:
- try-catch 错误捕获
- 模拟数据降级
- 用户友好提示

### 2. 图表展示
价格趋势图表:
- 自定义柱状图
- 点击交互效果
- 动态高度计算

### 3. 分类筛选
样品展示:
- 分类切换
- computed 过滤
- 条件渲染

### 4. 响应式布局
- 全部使用 rpx 单位
- 适配不同屏幕
- 弹性布局 (flexbox)
