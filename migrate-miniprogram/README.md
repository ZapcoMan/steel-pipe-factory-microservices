# 钢铁工厂小程序

一个专为钢铁工厂设计的微信小程序，提供生产管理、数据分析和计算工具等功能，帮助工厂提高生产效率和管理水平。

## 功能特性

### 📊 首页功能
- **品牌展示**：展示企业品牌信息和形象
- **主功能入口**：快速访问重量计算、价格行情、技术资料和样品展示功能
- **资讯滚动**：实时获取行业动态和资讯
- **合作企业**：展示精选合作企业信息
- **快捷操作**：便捷的联系和咨询服务入口

### 🔢 计算器功能
- **重量计算**：支持多种钢材规格的重量计算
- **计算器详情**：详细的计算公式和参数设置
- **历史记录**：保存计算历史，方便查询

### 💰 钢材价格行情
- **实时价格**：展示当前钢材市场价格
- **价格趋势**：提供价格趋势图表
- **多品类支持**：涵盖螺纹钢、线材、热轧板卷等多种钢材品类

### 📚 技术资料库
- **材质说明**：详细说明各种钢材材质特性
- **对比分析**：不同钢材材质的特性对比
- **应用场景**：提供使用场景建议和推荐

### 🖼️ 钢材样品展示
- **实物图片**：不同规格钢材的实物对比
- **详细参数**：规格、材质、特性等详细信息
- **分类展示**：按类型分类展示各类钢材样品

### ℹ️ 关于我们
- **企业介绍**：详细介绍企业背景和业务范围
- **联系方式**：提供联系电话、邮箱和地址
- **在线联系**：便捷的联系按钮，可直接拨打电话

## 技术架构

### 前端技术栈
- **框架**：Vue 3 + UniApp
- **状态管理**：Vue 3 Composition API
- **UI组件**：自定义组件库，适配小程序平台
- **样式**：使用 rpx 单位，适配不同屏幕尺寸

## 安装和运行

### 环境要求
- Node.js >= 14.0.0
- 微信开发者工具

### 安装步骤
1. 克隆项目到本地：
   ```bash
   git clone https://github.com/ZapcoMan/steel-factory-wxmini-program.git
   ```

2. 安装依赖：
   ```bash
   cd steel-factory-wxmini-program
   npm install # 或者
   pnpm install # 笔者语：我用的是 pnpm
   ```

3. 启动开发模式：
   ```bash
   npm run dev:mp-weixin # 或者
   pnpm run dev:mp-weixin 
   ```

4. 在微信开发者工具中打开项目目录

## API服务说明

项目中的API服务按功能分离，包括：
- **homeApi**：首页相关数据（资讯、合作企业等）
- **steelPriceApi**：钢材价格行情数据
- **techDataApi**：技术资料库数据
- **sampleApi**：样品展示数据

## 代码规范

- **命名规范**：采用驼峰命名法（camelCase）
- **组件命名**：采用帕斯卡命名法（PascalCase）
- **文件命名**：组件文件以.vue结尾，工具文件以.ts结尾
- **样式单位**：使用 rpx 适配不同屏幕尺寸
